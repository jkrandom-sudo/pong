# Pong 乒乓球 🏓

控制台版经典乒乓球游戏，使用 Python curses 实现的 TUI（终端用户界面）。

## 特性

- **两种游戏模式**：单人 vs AI、双人对战
- **AI 对手**：智能追踪球位置，带随机误差和延迟（可挑战但可击败）
- **中英文双语**：游戏内随时切换语言（C 键中文 / E 键英文）
- **完整菜单系统**：开始菜单、历史记录、游戏内帮助
- **速度控制**：按 +/- 调节球速（共 10 级）
- **音效反馈**：终端响铃 + 视觉闪烁，支持 3 种音效模式（关/仅视觉/视觉+响铃）
- **分数记录**：自动保存分数到 scores.json，支持历史排行榜（Top 10）
- **暂停/继续**：按 P 随时暂停
- **重新开始**：游戏结束后按 R 重开

## 操作说明

| 按键 | 功能 |
|------|------|
| W / S | 玩家1 上/下移动 |
| ↑ / ↓ | 玩家2 或 AI 对手 上/下移动 |
| P | 暂停/继续 |
| + / - | 增加/减少球速 |
| S | 切换音效模式（关→仅视觉→视觉+响铃） |
| C / E | 切换中文/英文 |
| H | 查看历史排行榜 |
| Q | 退出游戏 |
| R | 游戏结束后重新开始 |

## 开始游戏

```bash
cd ~/games/pong
python3 game.py
```

先选择语言（C=中文, E=English），然后在主菜单选择模式。

## 文件结构

```
pong/
├── game.py          # 主游戏文件（curses 循环 + 渲染）
├── game_logic.py    # 纯游戏逻辑（Ball, Paddle, GameState，无 curses 依赖）
├── scores.py        # 分数持久化（保存/加载/排行榜）
├── test_game.py     # 75 个自动化测试
├── scores.json      # 分数数据（自动生成）
└── README.md        # 本文件
```

## 技术栈

- **语言**：Python 3.11
- **UI 框架**：curses（标准库，无需额外安装）
- **测试框架**：pytest
- **代码行数**：game.py 586行 + game_logic.py 314行 + scores.py 110行 = ~1010行

## 测试

```bash
cd ~/games/pong
python3 -m pytest test_game.py -v
```

包含 75 个测试用例，覆盖：
- Ball 运动、弹跳、速度变化
- Paddle 移动、边界约束
- 碰撞检测（墙弹、球拍击中、球拍未中）
- 计分逻辑、胜利条件（11分）
- AI 追踪逻辑、延迟、随机误差
- 分数持久化（保存/加载/损坏文件处理）
- 边界情况（角落、快速操作、语言切换）
- 游戏状态管理（暂停、速度、音效模式）
